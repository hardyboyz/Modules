Work Order Tasks

Work order tasks are the building block of a work order template.

Adding a Work Order Task

Work order tasks are managed through the work order task screen. To add a task, follow the steps below:

  1. Navigate to Inventory -> Work Order Task. If this screen currently being viewed, click the New icon to clear the form.
  2. Fill in the fields as defined:
    Task Name: Contains a short title to use for the task. This title is used when building work order templates.
    Description: Contains a full description of thee task which will appear in the task list on the work order manager screen during the work order build process and on the work order form that can be printed with each work order.
    Department: Contains the responsible department for the task.
    Ref Documents: Lists any reference documents that provide additional direction/clarification/process control over the task.
    Ref Drawings: Lists any reference drawings that may provide additional direction/clarification/process control over the task.
    Task Time: Estimated/expected time to complete the task. this is used for statistical measurement of task execution time compared with actual execution time.
    Mfg Signoff: Whether or not a electronic signoff of the task is required by manufacturing personnel. If Yes, requires a user name and password to complete the task step and move to the next. If No, the step can be completed with a click of the Update button.
    QA signoff: Whether or not a electronic signoff of the task is required by Quality Assurance personnel. If Yes, requires a user name and password to complete the task step and move to the next.
    Data Entry: Whether or not data is required to be entered to complete the task step. Measurements, dimensions, lot codes, etc may be entered into the database for record keeping.
    ERP Entry: Determines whether to assemble the number of units into the ERP inventory system. NOTE: This should only be set to Yes once per work order template to prevent multiple entries into the ERP system.
  3. Click the Save icon to save the template and clear the form.

Editing a Work Order Task

Work order tasks can be edited at any time.

  1. Navigate to Inventory -> Work Order Tasks.
  2. Locate the task to be edited, either through the paging system or searching.
  3. Click on the row to be edited.
  4. Change the fields as necessary and click the Save icon to save the task and clear the form.

Deleting a Work Order Task

NOTE: Work order tasks can only be deleted if they have not been used in a Work Order.

  1. Navigate to Inventory -> Work Order Tasks.
  2. Locate the work order task to be deleted, either through the paging system or searching.
  3. Click on delete icon on  the work order task to be deleted.
  4. Confirm the deletion of the task.
