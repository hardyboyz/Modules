Work Order Builder

The Work Order Builder provides the ability to build templates from which items can be assembled. A work order template consists of one or more work order tasks in sequence to help control the assembly process as an integral part of a quality control system.

Adding a Work Order Template

Work order templates are managed through the work order Builder screen. To add a template, follow the steps below:

  1. Navigate to Inventory -> Work Order Builder. If the main screen currently being viewed, click the New icon .
  2. Fill in the fields as defined:
    Work Order Title: Contains a short title to use for the template. This title is used when generating work orders from the pop-up list.
    SKU: Enter a sku or select a sku from the Search pop-up list. Note, only skus that are assemblies will appear in the pop-up.
    Description: This field will be filled in automatically if the sku is selected from the pop-up window. It may be edited, left alone or re-written as necessary to describe the work order and display all pertinent information.
    Revision: This field is read-only and will be controlled by the module.
    Use Allocation: Determines whether or not to allocate parts with the work order. If Yes is selected, the quantity_on_allocation values will be adjusted for all parts required to build the assembly.
    Ref Documents: Lists any reference documents that provide additional direction/clarification/process control while assembling the sku.
    Ref Drawings: Lists any reference drawings that may provide additional direction/clarification/process control while assembling the sku.
  3. For all the task steps needed populate the procedure list to assemble the sku by selecting from the list of tasks created with the Work Order Task process. The sequence should be the same as what it would take to build the item.
  4. Click the Save icon to save the task and return to the main template list page.

Editing a Work Order Template

Work order templates can be edited at any time.

  1. Navigate to Inventory -> Work Order Builder.
  2. Locate the template to be edited, either through the paging system or searching.
  3. Click on the row to be edited. Note: Some fields will not be editable and are locked by the system. This is to maintain quality control over the work order process.
    REVISIONS: If the work order has been used to build a inventory item. The revision will automatically be rolled to the next level.
  4. Change the fields as necessary and click the Save icon to save the template and clear the form.

Deleting a Work Order Template

Work order templates can only be deleted if they have not been used in a Work Order.

  1. Navigate to Inventory -> Work Order Builder.
  2. Locate the template to be deleted, either through the paging system or searching.
  3. Click on Delete icon on the work order template to be deleted.
  4. Confirm the deletion of the template.
