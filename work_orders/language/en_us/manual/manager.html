Work Order Manager

The work order manager control the work order process through manufacturing. Work orders are specific to a sku and are reused over and over to manage production work flow, provide record keeping, track assembly times and manage inventory.

Creating a Work Order to Build Something

Before you can assemble an inventory item, a work order template must exist in the system.

  1. Navigate to Inventory -> Work Order Manager.
  2. Click on the New icon to start the create work order screen.
  3. Search for a sku from the pop-up. Only skus that have a work order template will appear in the pop-up. Click on the SKU you want to assemble.
  4. The SKU, Work Order Title will be filled in automatically. also, the task list will be displayed after the Comments field.
  5. Enter the quantity to assemble.
  6. If you use the priority indications, select a priority form the drop-down menu.
  7. If this work order requires any special instructions, enter them in the Work Order Notes and Comments field. Note that these comments can be added to, edited or removed during the assembly process.
  8. Click the Save icon (or Print and save icon ) to post (and print) the work order.

Editing a Work Order

A work order can be edited at any time. However, only some of the fields may be altered from when it was originally created.

  1. Navigate to Inventory -> Work Order Manager.
  2. Locate the work order to be edited, either through the paging system or searching.
  3. Click on the Edit icon on the action column of the work order to be edited. Note: Some fields will not be editable and are locked by the system. This is to maintain quality control over the work order process.
  4. Change the fields as necessary and click the Save icon to save the task and clear the form.

Executing the Steps of a Work Order

A work order must be acted upon to complete the process.  To act upon a work order:

  1. Navigate to Inventory -> Work Order Manager.
  2. Locate the work order to act upon and click on the work order line or the Build Work Order icon to open the build page.
  3. The task line that is being worked on will be highlighted. After completion of the step, enter the information, as requested by the task requirements, and press the Update button.
  4. After completion of all task steps of the work order, The work order will close and the page will return to the main work order manager page.

Deleting a Work Order

Work orders can be deleted at any time.

  1. Navigate to Inventory -> Work Order Manager.
  2. Locate the work order to be deleted, either through the paging system or searching.
  3. Click on Delete icon on the work order to be deleted.
  4. Confirm the deletion of the work order.
