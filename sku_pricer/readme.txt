SKU Pricer Module

Author: PhreSoft, LLC
Date: 2011-05-26
REvision: 1.0

Installation
1. Copy the SKUPricer folder to the /modules/ directory.
2. Navigate to Company -> Module administration and Install the SKUPricer Module.
3. Edit your user and set permissions to full for the module (under Tools tab).

Usage
1. Navigate to Tools -> SKU Pricer.
2. Select the file to update prices and press the Save icon.
3. The proces will be updated for the sku's that are found and a total count will be displayed when complete.
